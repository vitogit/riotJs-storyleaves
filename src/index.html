Acerca del proyecto

Esta pagina fue creada para poder jugar al juego StoryLeaves online como un proyecto personal, algunas reglas fueron modificadas.
StoryLeaves es un juego creado por Jamie Fristrom, las reglas en ingles se encuentran en http://www.gamedevblog.com/storyleaves.html
Las reglas en español se pueden encontrar en http://thredith.blogspot.com/2011/05/storyleaves-o-deshojacuentos-reglas.html (Nueva redacción por Thredith (8 págs)
Las cartas utilizadas son de http://labsk.net/wkr/?s=storyleaves , donde existe mucha información sobre el juego.

Fue programado en javascript utilizando el riot.js framework, y Mocha framework para testeo. Funciona en las ultimas versiones de Firefox y Chrome El código se encuentra en github https://github.com/vitogit/riotJs-storyleaves
